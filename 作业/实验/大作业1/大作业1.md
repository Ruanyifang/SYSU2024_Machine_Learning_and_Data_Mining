### 二、实验1：线性分类器与两种损失函数比较

#### 实验原理

1. ##### 线性分类器

   线性分类器的基本假设是数据是**线性可分**的。在二分类问题中，线性分类器的模型可以表示为： $y = \mathbf{w}^T \mathbf{x}$

   其中：

   - $\mathbf{w}$ 是模型的权重向量
   - $\mathbf{x}$ 是输入样本的特征向量

   模型通过调整权重 $\mathbf{w}$ 来学习如何对数据进行分类。不同的损失函数用来度量模型输出与真实标签之间的差距，并通过梯度下降法来优化该差距

   

2. ##### 损失函数

   损失函数是衡量模型预测与真实标签之间误差的函数。它决定了如何惩罚不准确的预测，从而指导模型的训练过程

   

   - **Hinge Loss**

     Hinge Loss 通常用于 **支持向量机 (SVM)**。它的目标是最大化分类的间隔，使得数据点不仅被正确分类，还要离决策边界尽可能远。其定义如下：
     $$
     L(y, f(\mathbf{x})) = \max(0, 1 - y f(\mathbf{x}))
     $$
     其中：

     - y 是真实标签
     - $f(\mathbf{x})$ 是模型的预测

     Hinge Loss 强制要求正确分类的点离决策边界至少距离 1，从而有助于间隔最大化

     

   - **Cross-Entropy Loss**

     Cross-Entropy Loss 通常用于 **逻辑回归**，广泛应用于二分类任务。其定义如下：
     $$
     L(y, p(\mathbf{x})) = - \left[ y \log(p(\mathbf{x})) + (1 - y) \log(1 - p(\mathbf{x})) \right]
     $$
     其中：

     -  $p(\mathbf{x})$ 是模型预测的概率

     交叉熵衡量了预测概率分布与真实标签之间的差异，越接近真实值损失越小

   

3. ##### 性能评估指标

   本次实验中，通过以下几个指标来评估模型的分类性能

   - **准确率 (Accuracy)**

     模型预测正确的样本数占总样本数的比例

   - **精确率 (Precision)**

     在所有预测为正类的样本中，实际为正类的样本所占的比例

   - **召回率 (Recall)**

     在所有实际为正类的样本中，被正确预测为正类的样本所占的比例

   - **F1 分数 (F1 Score)**

      精确率和召回率的调和平均数，综合衡量分类器的性能





#### 关键代码展示

```python
# 线性分类器
class LinearClassifier:
    def __init__(self, loss='hinge', learning_rate=0.01, n_epochs=1000):
        self.loss = loss
        self.learning_rate = learning_rate
        self.n_epochs = n_epochs
        self.weights = None

    def fit(self, X, y):
        num_samples, num_features = X.shape  # 获取样本数和特征数
        # 初始化权重为零
        self.weights = np.zeros(num_features)
        
        for epoch in range(self.n_epochs):
            for i in range(num_samples):
                xi = X[i]  # 当前样本的特征
                yi = y[i]  # 当前样本的标签
                # 计算预测值
                prediction = np.dot(xi, self.weights)  # 线性组合

                if self.loss == 'hinge':
                    # Hinge Loss的实现
                    # 如果预测的结果不满足条件，则更新权重
                    if yi * prediction < 1:
                        self.weights += self.learning_rate * (yi * xi - 2 * self.weights)
                    else:
                        self.weights -= self.learning_rate * 2 * self.weights
                
                elif self.loss == 'cross_entropy':
                    # Cross-Entropy Loss的实现
                    prob = 1 / (1 + np.exp(-prediction))  # Sigmoid函数得到概率
                    gradient = prob - yi  # 计算梯度
                    self.weights -= self.learning_rate * gradient * xi  # 更新权重

    def predict(self, X):
        predictions = np.dot(X, self.weights)
        return (predictions > 0).astype(int)  # 将预测结果转换为0或1

# 评估模型性能
def evaluate_model(y_true, y_pred):
    accuracy = accuracy_score(y_true, y_pred)
    precision = precision_score(y_true, y_pred)
    recall = recall_score(y_true, y_pred)
    f1 = f1_score(y_true, y_pred)
    return accuracy, precision, recall, f1
```

### 三、实验2：多项式分类器，不同多项式阶数下的表现及过拟合现象的观察

#### 实验原理

##### 多项式回归

多项式回归是对线性回归的一种扩展，用来拟合非线性关系的数据。通过将输入特征 x 转换为多项式形式，即 $x^d$ 的形式，多项式回归可以拟合更加复杂的曲线。因此，通过调整多项式的次数 d，可以控制模型的复杂度。过高的次数往往会导致过拟合，表现为模型对训练数据拟合得过于精准，缺乏泛化能力





#### 关键代码展示


### 四、使用线性核函数和高斯核函数的SVM

#### 实验原理

1. ##### 基本概念

   SVM是一种监督学习模型，主要用于二分类任务。它通过在高维空间中寻找一个**最优的超平面**将数据集的**不同类别进行分离**。SVM的目标是找到间隔最大的决策边界（即超平面），并且**最大化类间的距离**，从而增强模型的泛化能力

   

2. ##### 超平面和分类

   在二维平面上，超平面就是一条直线，在三维空间上是一个平面。而在高维空间中，超平面是维度减1的一个几何对象。对于一个给定的数据集，SVM通过公式构造超平面：$\mathbf{w} \cdot \mathbf{x} + b = 0$ 其中，$\mathbf{w}$ 是超平面的法向量，决定了超平面的方向；b 是偏置项，控制超平面的位置

   

3. ##### 间隔最大化

   SVM的核心思想是**间隔最大化**

   对于给定的超平面，**距离超平面最近的样本点**称为**支持向量**，这些点**决定了超平面的最终位置**。它们在训练模型时起关键作用，而其他远离超平面的点则对超平面的确定没有影响

   SVM在训练过程中不仅寻找能正确分类数据的超平面，还要求这个超平面与支持向量之间的距离（间隔）最大化，从而使模型对数据有更好的泛化能力

   

4. ##### 软间隔和惩罚参数C

   在真实问题中，数据通常是线性不可分的，SVM通过引入**软间隔**来处理这种情况。软间隔允许一些数据点可以位于错误的一侧，但会对错误分类施加惩罚。惩罚程度由参数**C**决定：

   - 大的C值会强制模型尽可能正确分类所有训练样本，但可能导致过拟合
   - 小的C值则允许更多的分类错误，但有更好的泛化能力

   

5. #####  核函数

   核函数本质上是一个**相似度函数**，它接收两个输入样本的特征向量 $\mathbf{x}$ 和 $\mathbf{x'}$，并输出一个表示它们相似度的数值。SVM不仅可以在线性可分的数据上使用，还可以通过**核技巧**扩展到非线性问题。核函数通过将低维数据映射到高维空间，使得在高维空间中可以找到线性可分的超平面。

   本次实验使用的核函数如下：

   - **线性核**

     适用于线性可分的数据，公式为 $\mathbf{k}(\mathbf{x}, \mathbf{x'}) = \mathbf{x} \cdot \mathbf{x'}$

   - **高斯核（RBF核）**

     常用于非线性可分的数据，公式为 $\mathbf{k}(\mathbf{x}, \mathbf{x'}) = \exp(-\gamma ||\mathbf{x} - \mathbf{x'}||^2)$，$\gamma$控制高斯分布的宽度

   

6. ##### SVM的优化目标

   SVM的目标是通过最小化以下损失函数来实现间隔最大化： 
   $$
   \min \frac{1}{2} ||\mathbf{w}||^2 + C \sum_{i=1}^n \max(0, 1 - y_i (\mathbf{w} \cdot \mathbf{x_i} + b))
   $$
    其中，前半部分是用来最大化间隔的，后半部分是对错误分类点的惩罚

   

7. ##### 线性核和高斯核（RBF核）对比

   - **线性核函数**

     适用于数据线性可分的情况。它简单且计算效率高，但在数据线性不可分的情况下性能较差

   - **高斯核（RBF核）**

     是非线性问题中常用的核函数。它通过将数据映射到高维空间，使得即使在低维空间中不可分的数据，也可能在高维空间中可分


     - `kernel='linear'`：指定使用线性核函数
     - `C=C_linear`：将惩罚参数C设置为1.0，以控制分类的复杂性

2. ##### 超参数的选择

   - 惩罚参数C
     - 控制模型对分类错误的容忍度
     - 较小的C值会导致更强的正则化，可能导致欠拟合；而较大的C值则可能导致模型过拟合
   - 高斯核的γ参数
     - 控制高斯核的宽度，影响决策边界的复杂性
     - 较小的$\gamma$会导致较平滑的决策边界，可能导致欠拟合；较大的$\gamma$会导致更复杂的决策边界，可能导致过拟合
     - 在代码中使用`'scale'`，可以自动根据数据的特征标准差来调整$\gamma$

   

3. ##### 使用的训练技巧


     在这个过程中，模型会根据训练集中的特征和目标值调整其参数，以最大化间隔并最小化分类错误
